# Badger

**All Your Achievements In One Place**  
From coding platforms to learning hubs – **Orbito** unifies your badges, so you can showcase your skills without chaos.

🌐 [Visit Orbito](https://orbito.space)

---

## 🚀 Overview
Badger is a platform designed to aggregate and display your achievements across multiple coding platforms and learning hubs. Instead of scattered badges and certificates, Badger centralizes them into a single, organized profile.

---

## ✨ Features
- **Unified Dashboard**: Collect badges from coding platforms, online courses, and learning hubs.  
- **Seamless Integration**: Connect with Orbito to sync your achievements.  
- **Profile Showcase**: Share your consolidated achievements with peers, recruiters, or communities.  
- **Modern Tech Stack**: Built with Node.js and Webpack for scalability and performance.  
- **Secure Authentication**: Logout and routing fixes ensure smooth and safe user experience.  

---

## 🛠️ Tech Stack
- **Frontend**: JavaScript (58.2%)  
- **Backend**: Node.js (30.4%)  
- **Other**: Additional utilities (10.8%)
