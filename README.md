#  Badger

A modular, privacy-conscious UI component library built with performance and reusability in mind. Designed to accelerate frontend development with clean, customizable building blocks.

## 🚀 Features

- ⚛️ React-based components with minimal dependencies
- 🎨 Easily themeable and extendable
- 🧩 Component-first structure for rapid prototyping
- 📦 Lightweight and tree-shakable
